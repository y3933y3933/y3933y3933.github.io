身為一個前端開發人員，很想使用自己熟悉的編輯器去開發 C#，本篇就來記錄如何在 visual code 上建置 C#開發環境。

# 前置作業
## 安裝VS Code
至[官網](https://code.visualstudio.com/)下載.exe執行檔，使用預設選項安裝就行。
![image](https://user-images.githubusercontent.com/28300286/234038454-095a2ba1-cb71-45cc-810f-d2820850583f.png)


## 安裝 C#擴充套件
安裝完後開啟VsCode，左側列有一個名叫Extensions的Icon，點擊後會出現你已安裝的擴充套件及推薦擴充套件，可以在最上方搜尋C#，安裝下方兩個擴充套件(如圖)
- [C# by Microsoft](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp)
- [C# Etensions by JosKreativ](https://marketplace.visualstudio.com/items?itemName=kreativ-software.csharpextensions)

![image](https://user-images.githubusercontent.com/28300286/234039890-ef0f3bb8-5420-4bdd-9698-a9f1bd90b390.png)

## 安裝.NET SDK
至[官網](https://dotnet.microsoft.com/en-us/download)安裝最新版SDK(當前最新版是.NET 7.0)，一樣使用預設選項安裝即可。
![image](https://user-images.githubusercontent.com/28300286/234040483-95e3f772-6ef0-4af1-a803-aff16a0d5379.png)

### 檢查是否安裝成功
在命令提示字元(cmd)中輸入以下指令，如安裝正確會顯示你所安裝的.NET SDK版本。
```
dotnet --version
```
![image](https://user-images.githubusercontent.com/28300286/234041139-f4aaa7e5-7ddc-4ae8-869c-e4d8bf833eb8.png)

---

# 執行程式碼
在cmd中，切換到所要建立專案的路徑下，執行以下指令：
```
dotnet new console
```

上述指令為dotnet CLI指令，會幫你建立出一個Console Application，建立成功會在目錄看到以下資料：
![image](https://user-images.githubusercontent.com/28300286/234570737-21607d65-cb88-499e-b13d-77d74fd1e13e.png)
<br/>
點開`Program.cs`會看到一行程式碼
![image](https://user-images.githubusercontent.com/28300286/234571346-6906d8a9-89e1-403c-9c8c-6ba6c2047daf.png)
<br/>
在cmd執行以下指令，就會印出Hello World!👏👏👏
```
dotnet run 
```
